JavaScript

Uma das linguagens mais populares do mundo para desenvolvimento web, responsável por interatividade em sites, aplicações modernas e sistemas completos no frontend e backend.

O que é JavaScript?

JavaScript é uma linguagem de programação interpretada usada principalmente para criar interatividade em páginas web. Ela roda diretamente no navegador e também pode ser usada no servidor com Node.js.

  • ✔ Linguagem do Frontend
  • ✔ Usada em sites e apps web
  • ✔ Pode rodar no servidor (Node.js)
  • ✔ Extremamente versátil

Onde JavaScript é usado?

🌐

Interatividade em sites e aplicações web.

⚙️

Backend
