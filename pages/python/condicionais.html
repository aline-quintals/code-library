Condicionais em Python

Condicionais permitem que o programa tome decisões com base em condições.

O que são condicionais?

As condicionais verificam se algo é verdadeiro ou falso para executar diferentes ações.

  • ✔ if
  • ✔ else
  • ✔ elif

Operadores de comparação

Os operadores são usados para comparar valores.

  • == Igual
  • != Diferente
  • > Maior que
  • < Menor que
  • >= Maior ou igual
  • <= Menor ou igual

If

* O if executa um bloco de código quando a condição for verdadeira. 


idade = 18

if idade >= 18:
    print("Maior de idade")

Saída:

Maior de idade

If e Else

* O else executa outro bloco caso a condição seja falsa. 


numero = 5

if numero % 2 == 0:
    print("Par")
else:
    print("Ímpar")

Saída:

Ímpar

If, Elif e Else

* O elif permite verificar novas condições caso a anterior seja falsa. 


nota = 7

if nota >= 9:
    print("Excelente")

elif nota >= 6:
    print("Aprovado")

else:
    print("Reprovado")

Saída:

Aprovado

Comando condicional

* O input() permite que o usuário digite informações no programa. 


numero = int(input("Digite um número: "))

if numero > 10:
    print("O número é maior que 10")

else:
    print("O número é menor ou igual a 10")

Exemplo:

Digite um número: 15
O número é maior que 10

Verificando senha

* O operador == verifica se dois valores são iguais. 


senha = input("Digite a senha: ")

if senha == "python123":
    print("Acesso permitido")

else:
    print("Senha incorreta")

Exemplo:

Digite a senha: python123
Acesso permitido

Usando AND

* O and exige que todas as condições sejam verdadeiras. 


idade = 20
tem_ingresso = True

if idade >= 18 and tem_ingresso:
    print("Entrada permitida")
else:
    print("Entrada negada")

Saída:

Entrada permitida

Usando OR

* O or executa o código se pelo menos uma condição for verdadeira. 


dia = "domingo"

if dia == "sábado" or dia == "domingo":
    print("Fim de semana")
else:
    print("Dia útil")

Saída:

Fim de semana

Dicas importantes

  • ✔ Use dois pontos : após o if, elif e else
  • ✔ A indentação é obrigatória no Python
  • ✔ Use TAB ou 4 espaços para organizar o código
  • ✔ As condições retornam verdadeiro ou falso
