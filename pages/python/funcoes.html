Funções em Python

Funções servem para organizar e reutilizar códigos.

O que são funções?

Funções agrupam instruções para serem reutilizadas várias vezes.

  • ✔ Organização
  • ✔ Reutilização
  • ✔ Código mais limpo

Função simples


def saudacao():
    print("Olá!")

saudacao()

Saída:

Olá!

Função com parâmetro


def mostrar_nome(nome):
    print(nome)

mostrar_nome("Aline")

Saída:

Aline
