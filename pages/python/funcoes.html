Funções em Python

Funções servem para organizar e reutilizar códigos.

O que são funções?

Funções agrupam instruções para serem reutilizadas várias vezes.

  • ✔ Organização
  • ✔ Reutilização
  • ✔ Código mais limpo

Estrutura de uma função

As funções em Python são criadas usando a palavra def. 


def nome_da_funcao():
    print("Olá")

Função simples

* A função só será executada quando for chamada pelo nome. 


def saudacao():
    print("Olá!")

saudacao()

Saída:

Olá!

Função com parâmetro

* Parâmetros permitem enviar informações para a função. 


def mostrar_nome(nome):
    print(nome)

mostrar_nome("Aline")

Saída:

Aline

Função com dois parâmetros

* Uma função pode receber mais de um parâmetro. 


def soma(a, b):
    print(a + b)

soma(5, 3)

Saída:

8

Função com retorno

* O return devolve um valor para ser utilizado depois. 


def multiplicar(a, b):
    return a * b

resultado = multiplicar(4, 5)

print(resultado)

Saída:

20

Função com input

* Funções também podem trabalhar com dados digitados pelo usuário. 


def mensagem():
    nome = input("Digite seu nome: ")
    print("Olá,", nome)

mensagem()

Exemplo:

Digite seu nome: Aline
Olá, Aline

Chamando a função várias vezes

* Uma função pode ser reutilizada várias vezes no programa. 


def saudacao():
    print("Bem-vindo!")

saudacao()
saudacao()
saudacao()

Saída:

Bem-vindo!
Bem-vindo!
Bem-vindo!

Dicas importantes 💡

  • ✔ Use def para criar funções
  • ✔ Funções ajudam na organização do código
  • ✔ Parâmetros recebem informações
  • ✔ O return devolve valores
  • ✔ Funções evitam repetição de código
