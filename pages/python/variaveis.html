Variáveis em Python

Variáveis em Python são utilizadas para armazenar informações e valores durante a execução do programa.

O que são variáveis?

Variáveis funcionam como espaços de memória que guardam informações utilizadas pelo sistema.

Em Python, não é necessário definir o tipo da variável. A própria linguagem identifica automaticamente.

  • ✔ Fácil de utilizar
  • ✔ Sintaxe simples
  • ✔ Tipagem dinâmica
  • ✔ Muito usada em automação e IA

Tipos mais utilizados

🔤

String

Armazena textos.

🔢

int

Números inteiros.

📈

float

Números decimais.

✔

bool

Verdadeiro ou falso.

String


nome = "Aline"

print(nome)

Saída:

Aline

Inteiro


idade = 20

print(idade)

Saída:

20

Float


altura = 1.70

print(altura)

Saída:

1.7

Boolean


aprovado = True

print(aprovado)

Saída:

True
