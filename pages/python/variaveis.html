Variáveis em Python

Variáveis em Python são utilizadas para armazenar informações e valores durante a execução do programa.

O que são variáveis?

Variáveis funcionam como espaços de memória que guardam informações utilizadas pelo sistema.

Em Python, não é necessário definir o tipo da variável. A própria linguagem identifica automaticamente.

  • ✔ Fácil de utilizar
  • ✔ Sintaxe simples
  • ✔ Tipagem dinâmica
  • ✔ Muito usada em automação e IA

Regras para criar variáveis

  • ✔ O nome não pode começar com números
  • ✔ Não pode conter espaços
  • ✔ Use nomes fáceis de entender
  • ✔ Python diferencia letras maiúsculas e minúsculas

nome = "Aline"
idade = 20

Tipos mais utilizados

🔤

String

Armazena textos.

🔢

int

Números inteiros.

📈

float

Números decimais.

✔

bool

Verdadeiro ou falso.

String

* Strings armazenam textos e devem ficar entre aspas. 


nome = "Aline"

print(nome)

Saída:

Aline

Inteiro

* O tipo int armazena números inteiros. 


idade = 20

print(idade)

Saída:

20

Float

* O tipo float armazena números com casas decimais. 


altura = 1.70

print(altura)

Saída:

1.7

Boolean

* O tipo bool possui apenas dois valores: True ou False. 


aprovado = True

print(aprovado)

Saída:

True

Juntando textos

* É possível juntar textos usando o símbolo +. 


nome = "Aline"

print("Olá, " + nome)

Saída:

Olá, Aline

Usando input()

* O input() permite que o usuário digite informações. 


nome = input("Digite seu nome: ")

print(nome)

Exemplo:

Digite seu nome: Aline
Aline

Alterando valores

* O valor de uma variável pode ser alterado a qualquer momento. 


cidade = "São Paulo"

cidade = "Rio de Janeiro"

print(cidade)

Saída:

Rio de Janeiro

* O comando type() mostra o tipo da variável.
