O que é Python?

Python é uma linguagem de programação criada por Guido van Rossum e lançada em 1991.

Ela ficou famosa por possuir uma sintaxe simples, limpa e fácil de aprender.

Python foi desenvolvida com foco em:

  • ✔ Facilidade de leitura
  • ✔ Produtividade
  • ✔ Código simples e organizado
  • ✔ Desenvolvimento rápido
