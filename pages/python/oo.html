Orientação a Objetos

A programação orientada a objetos organiza o código em objetos e classes.

O que é POO?

POO significa Programação Orientada a Objetos.

  • ✔ Classes
  • ✔ Objetos
  • ✔ Métodos
  • ✔ Atributos

Criando uma classe


class Pessoa:

    def __init__(self, nome):
        self.nome = nome

pessoa = Pessoa("Aline")

print(pessoa.nome)

Saída:

Aline

Método


class Cachorro:

    def latir(self):
        print("Au au!")

dog = Cachorro()

dog.latir()

Saída:

Au au!
