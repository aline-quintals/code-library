Orientação a Objetos

A programação orientada a objetos organiza o código em objetos e classes.

O que é POO?

POO significa Programação Orientada a Objetos.

  • ✔ Classes
  • ✔ Objetos
  • ✔ Métodos
  • ✔ Atributos

Conceitos importantes

📦

Classe

É o modelo utilizado para criar objetos.

👤

Objeto

É a instância criada a partir da classe.

⚙️

Método

É uma função criada dentro da classe.

📝

Atributo

São as características do objeto.

Estrutura de uma classe

As classes são criadas usando a palavra class. 


class Pessoa:
    pass

O comando pass serve para criar uma classe vazia temporariamente.

Criando uma classe

* A função __init__() é executada automaticamente quando o objeto é criado. 


class Pessoa:

    def __init__(self, nome):
        self.nome = nome

pessoa = Pessoa("Aline")

print(pessoa.nome)

Saída:

Aline

Criando objetos

* Objetos são criados a partir de uma classe. 


class Carro:

    def __init__(self, modelo):
        self.modelo = modelo

carro1 = Carro("Ferrari")
carro2 = Carro("BMW")

print(carro1.modelo)
print(carro2.modelo)

Saída:

Ferrari
BMW

Método

* Métodos são funções que pertencem à classe. 


class Cachorro:

    def latir(self):
        print("Au au!")

dog = Cachorro()

dog.latir()

Saída:

Au au!

Atributos

* Atributos armazenam informações do objeto. 


class Aluno:

    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

aluno = Aluno("Carlos", 16)

print(aluno.nome)
print(aluno.idade)

Saída:

Carlos
16

Usando SELF

* O self representa o próprio objeto criado. 


class Pessoa:

    def falar(self):
        print("Olá!")

p = Pessoa()

p.falar()

Saída:

Olá!

Alterando atributos

* Os atributos podem ser modificados depois da criação do objeto. 


class Jogo:

    def __init__(self, nome):
        self.nome = nome

game = Jogo("Minecraft")

game.nome = "Roblox"

print(game.nome)

Saída:

Roblox

Método com parâmetro

* Métodos também podem receber parâmetros. 


class Calculadora:

    def soma(self, a, b):
        print(a + b)

calc = Calculadora()

calc.soma(5, 8)

Saída:

13

Exemplo do mundo real 🌎

Uma classe funciona como uma forma ou modelo.

  • ✔ Classe: Carro
  • ✔ Objeto: Ferrari
  • ✔ Atributo: cor
  • ✔ Método: acelerar()

Dicas importantes 💡

  • ✔ Classes criam objetos
  • ✔ Objetos possuem atributos e métodos
  • ✔ O self representa o próprio objeto
  • ✔ O __init__() inicia os atributos
  • ✔ POO ajuda na organização do código
  • ✔ Classes podem ser reutilizadas várias vezes
