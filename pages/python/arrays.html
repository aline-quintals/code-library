Arrays em Python

Arrays e listas armazenam múltiplos valores.

O que são arrays?

Python utiliza principalmente listas para armazenar conjuntos de dados.

  • ✔ Armazenam vários valores
  • ✔ Permitem repetição
  • ✔ Podem ser percorridas com loops

Lista simples


frutas = ["maçã", "banana", "uva"]

print(frutas)

Saída:

['maçã', 'banana', 'uva']

Percorrendo lista


nomes = ["Ana", "Carlos", "Aline"]

for nome in nomes:
    print(nome)

Saída:

Ana
Carlos
Aline
