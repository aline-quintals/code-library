Arrays em Python

Arrays e listas armazenam múltiplos valores.

O que são arrays?

Python utiliza principalmente listas para armazenar conjuntos de dados.

  • ✔ Armazenam vários valores
  • ✔ Permitem repetição
  • ✔ Podem ser percorridas com loops
  • ✔ Aceitam diferentes tipos de dados

Estrutura de uma lista

As listas usam colchetes [] e os valores são separados por vírgulas. 


cores = ["Azul", "Verde", "Vermelho"]

Lista simples

* Uma lista pode armazenar vários valores dentro de uma única variável. 


frutas = ["maçã", "banana", "uva"]

print(frutas)

Saída:

['maçã', 'banana', 'uva']

Percorrendo lista

* O for percorre cada item da lista automaticamente. 


nomes = ["Ana", "Carlos", "Aline"]

for nome in nomes:
    print(nome)

Saída:

Ana
Carlos
Aline

Lista de frutas

* A variável fruta recebe um item da lista por vez. 


frutas = ["Maçã", "Banana", "Uva", "Morango", "Laranja"]

for fruta in frutas:
    print(fruta)

Saída:

Maçã
Banana
Uva
Morango
Laranja

Lista de maquiagem

* Listas podem armazenar qualquer tipo de informação, como objetos, nomes ou produtos. 


maquiagem = ["Blush", "Pó", "Rímel", "Base", "Batom"]

for item in maquiagem:
    print(item)

Saída:

Blush
Pó
Rímel
Base
Batom

Adicionando itens

* O método append() adiciona um novo item ao final da lista. 


frutas = ["Maçã", "Banana"]

frutas.append("Uva")

print(frutas)

Saída:

['Maçã', 'Banana', 'Uva']

Acessando posições

* As listas começam na posição 0. 


cores = ["Azul", "Verde", "Vermelho"]

print(cores[0])
print(cores[1])

Saída:

Azul
Verde

Lista vazia com FOR

* É possível criar uma lista vazia e adicionar valores usando append(). 


numeros = []

for i in range(5):
    numero = int(input("Digite um número: "))
    numeros.append(numero)

print(numeros)

Exemplo:

Digite um número: 2
Digite um número: 4
Digite um número: 6
Digite um número: 8
Digite um número: 10

[2, 4, 6, 8, 10]

Tamanho da lista

* O len() mostra quantos itens existem na lista. 


animais = ["Cachorro", "Gato", "Coelho"]

print(len(animais))

Saída:

3

Dicas importantes

  • ✔ As listas começam no índice 0
  • ✔ Use append() para adicionar itens
  • ✔ O for ajuda a percorrer listas
  • ✔ Use len() para descobrir o tamanho da lista
  • ✔ Listas podem guardar textos, números e booleanos
