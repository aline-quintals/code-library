Loops em Python

Loops são estruturas de repetição utilizadas para executar ações várias vezes de forma automática.

O que são loops?

Em Python, loops permitem repetir blocos de código enquanto uma condição for verdadeira ou enquanto existirem itens para percorrer.

Eles são muito utilizados para:

  • ✔ Repetir tarefas automáticas
  • ✔ Percorrer listas
  • ✔ Criar contadores
  • ✔ Fazer verificações

Tipos de loops

🔄

for

Percorre listas, textos e sequências.

♾️

while

Repete enquanto a condição for verdadeira.

📋

range()

Cria sequências numéricas.

🛑

break

Interrompe um loop.

Loop For

* O for percorre cada item da lista automaticamente. 


frutas = ["maçã", "banana", "uva"]

for fruta in frutas:
    print(fruta)

Saída:

maçã
banana
uva

For percorrendo lista

* A variável fruta recebe um item da lista por vez. 


frutas = ["banana", "maçã", "pera", "abacaxi", "laranja"]

for fruta in frutas:
    print(fruta)

Resultado:

banana
maçã
pera
abacaxi
laranja

For percorrendo texto

* O for também pode percorrer letras de uma palavra. 


for letra in "banana":
    print(letra)

Resultado:

b
a
n
a
n
a

Loop While

* O while continua executando enquanto a condição for verdadeira. 


contador = 1

while contador <= 5:
    print(contador)

    contador += 1

Saída:

1
2
3
4
5

Usando break

* O break interrompe o loop imediatamente. 


i = 1

while i < 6:

    print(i)

    if i == 3:
        break

    i += 1

Resultado:

1
2
3

Usando range()

* O range() cria uma sequência de números. 


for x in range(1, 5):
    print(x)

Resultado:

1
2
3
4

Range com passo

* O terceiro valor do range() define o passo da contagem. 


for numero in range(0, 11, 2):
    print(numero)

Resultado:

0
2
4
6
8
10

⚠ Cuidado com loops infinitos

* Um loop infinito acontece quando a condição nunca deixa de ser verdadeira. 


while True:
    print("Esse loop nunca termina")

Use com cuidado para evitar travamentos.

Dicas importantes 💡

🔁 Use for para listas e sequências
⚠ Evite loops infinitos no while
📚 range() ajuda em contagens
🚀 Loops deixam o código automático
🛑 break interrompe a repetição
🔤 O for também percorre textos
