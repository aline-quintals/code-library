SELECT

O comando SELECT é utilizado para buscar informações dentro do banco de dados.

O que o SELECT faz?

O SELECT consulta dados armazenados nas tabelas.

  • ✔ Buscar registros
  • ✔ Filtrar informações
  • ✔ Ordenar resultados
  • ✔ Exibir colunas específicas
