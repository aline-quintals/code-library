CREATE TABLE

O CREATE TABLE é utilizado para criar novas tabelas dentro do banco de dados.

O que é uma tabela?

As tabelas armazenam os dados do banco. Cada tabela possui colunas e linhas.

  • ✔ Colunas representam os tipos de dados
  • ✔ Linhas representam os registros
  • ✔ Cada tabela possui um nome
  • ✔ É possível definir chaves primárias

Criando uma tabela


CREATE TABLE alunos (

    id INT PRIMARY KEY,

    nome VARCHAR(100),

    idade INT

);

Resultado:

Tabela "alunos" criada com sucesso.

Tipos de dados comuns

🔢

INT

Armazena números inteiros.

🔤

VARCHAR

Armazena textos.

📅

DATE

Armazena datas.

✔

BOOLEAN

Armazena verdadeiro ou falso.
