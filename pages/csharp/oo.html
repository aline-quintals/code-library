Orientação a Objetos em C#

C# é uma linguagem fortemente orientada a objetos.

O que é POO?

POO organiza o código usando classes e objetos para representar o mundo real.

  • ✔ Classes
  • ✔ Objetos
  • ✔ Métodos
  • ✔ Atributos

Criando uma classe

* Uma classe é um modelo para criar objetos. 


class Pessoa
{
    public string nome = "Aline";
}

Pessoa p = new Pessoa();

Console.WriteLine(p.nome);

Método

* Métodos representam ações de um objeto. 


class Cachorro
{
    public void Latir()
    {
        Console.WriteLine("Au au!");
    }
}

Cachorro c = new Cachorro();
c.Latir();

Dicas importantes 💡

🧠 Classes são modelos de objetos
📦 Objetos são instâncias da classe
⚙️ Métodos representam ações
🏷️ Atributos armazenam dados
🚀 POO é essencial em projetos grandes
🔒 Ajuda na organização e manutenção do código
