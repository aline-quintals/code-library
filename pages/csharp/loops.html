Loops em C#

Loops são estruturas que repetem blocos de código automaticamente.

O que são loops?

Em C#, loops permitem executar uma ação várias vezes enquanto uma condição for verdadeira ou enquanto houver itens para percorrer.

  • ✔ for
  • ✔ while
  • ✔ foreach

Loop for

* O for é usado quando sabemos exatamente quantas vezes queremos repetir algo. 


for (int i = 1; i <= 5; i++)
{
    Console.WriteLine(i);
}

Saída:

1
2
3
4
5

Loop while

* O while repete enquanto a condição for verdadeira. 


int i = 1;

while (i <= 5)
{
    Console.WriteLine(i);
    i++;
}

Saída:

1
2
3
4
5

Loop foreach

* O foreach percorre listas e arrays automaticamente. 


string[] nomes = { "Ana", "Carlos", "Aline" };

foreach (string nome in nomes)
{
    Console.WriteLine(nome);
}

Saída:

Ana
Carlos
Aline

Dicas importantes 💡

🔁 Use for quando souber o número de repetições
⚠ while pode gerar loop infinito se não tomar cuidado
📚 foreach é ideal para arrays e listas
🚀 Loops ajudam a automatizar tarefas repetitivas
🧠 Sempre atualize variáveis no while
