Condicionais em C#

Condicionais permitem que o programa tome decisões com base em condições.

O que são condicionais?

Em C#, usamos estruturas condicionais para executar diferentes blocos de código dependendo de uma condição.

  • ✔ if
  • ✔ else
  • ✔ else if

if simples

* O if executa o código apenas se a condição for verdadeira. 


int idade = 18;

if (idade >= 18)
{
    Console.WriteLine("Maior de idade");
}

if e else

* O else executa quando a condição do if não é verdadeira. 


int numero = 5;

if (numero % 2 == 0)
{
    Console.WriteLine("Par");
}
else
{
    Console.WriteLine("Ímpar");
}

Dicas importantes 💡

🔀 Use if para decisões simples
⚡ else executa quando o if falha
🧠 else if ajuda em múltiplas condições
🚀 Condicionais são base da lógica de programação
📌 Sempre use operadores corretos (==, !=, >, <)
