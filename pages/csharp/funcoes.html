Funções em C#

Funções (métodos) são blocos de código reutilizáveis.

O que são funções?

Em C#, funções são chamadas de métodos e ajudam a organizar o código.

  • ✔ Reutilização
  • ✔ Organização
  • ✔ Código limpo

Método simples

* Um método executa uma ação quando chamado. 


static void Saudacao()
{
    Console.WriteLine("Olá!");
}

Saudacao();

Método com parâmetro

* Parâmetros permitem enviar informações para o método. 


static void MostrarNome(string nome)
{
    Console.WriteLine(nome);
}

MostrarNome("Aline");

Dicas importantes 💡

⚙️ Funções ajudam a evitar repetição de código
📦 Métodos devem ter uma responsabilidade única
🚀 Use parâmetros para deixar funções mais flexíveis
🧠 Evite funções muito grandes
📚 Reutilização melhora organização do código
