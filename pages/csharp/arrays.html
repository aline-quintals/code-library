Arrays em C#

Arrays armazenam vários valores do mesmo tipo em uma única variável.

O que são arrays?

Arrays em C# são estruturas que armazenam múltiplos valores organizados por índice.

  • ✔ Vários valores
  • ✔ Índices começam em 0
  • ✔ Organização de dados

Array simples

* Cada posição do array possui um índice. 


string[] frutas = { "Maçã", "Banana", "Uva" };

Console.WriteLine(frutas[0]);

Loop no array

* Podemos percorrer arrays com foreach. 


string[] frutas = { "Maçã", "Banana", "Uva" };

foreach (string fruta in frutas)
{
    Console.WriteLine(fruta);
}

Dicas importantes 💡

📚 Arrays começam no índice 0
⚠ Acesso inválido pode gerar erro (IndexOutOfRange)
🔁 Use foreach para percorrer facilmente
🚀 Arrays ajudam a organizar dados do mesmo tipo
🧠 Use Length para saber o tamanho
