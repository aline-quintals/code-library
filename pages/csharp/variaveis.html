Variáveis em C#

Variáveis são espaços de memória utilizados para armazenar dados que serão usados durante a execução do programa em C#.

O que são variáveis?

Em C#, variáveis são utilizadas para guardar valores como textos, números e valores lógicos. Diferente do Python, o C# exige que o tipo seja declarado.

Cada variável em C# possui:

  • ✔ Um tipo (int, string, bool...)
  • ✔ Um nome
  • ✔ Um valor

Regras das variáveis

O C# possui regras importantes para garantir organização e segurança no código.

  • ✔ O tipo deve ser declarado obrigatoriamente
  • ✔ O nome não pode começar com número
  • ✔ C# diferencia maiúsculas e minúsculas
  • ✔ Toda instrução termina com ;

Tipos mais utilizados

🔤

string

Armazena textos.

🔢

int

Números inteiros.

📈

double

Números decimais.

✔

bool

true ou false.

🔡

char

Um único caractere.

String

* O tipo string é usado para armazenar textos em C#. Os valores devem estar entre aspas duplas. 


string nome = "Aline";

Console.WriteLine(nome);

Saída:

Aline

Inteiro

* O tipo int armazena números inteiros sem casas decimais. 


int idade = 20;

Console.WriteLine(idade);

Saída:

20

Double

* O double é usado para números com casas decimais, como altura, peso e valores precisos. 


double altura = 1.70;

Console.WriteLine(altura);

Saída:

1.7

Boolean

* O tipo bool armazena apenas dois valores: true ou false. 


bool aprovado = true;

Console.WriteLine(aprovado);

Saída:

True

Char

* O tipo char armazena apenas um único caractere usando aspas simples. 


char letra = 'A';

Console.WriteLine(letra);

Saída:

A

Alterando valores

* Variáveis podem ter seus valores alterados durante o programa. 


string cidade = "São Paulo";

cidade = "Rio de Janeiro";

Console.WriteLine(cidade);

Saída:

Rio de Janeiro

Constantes (const)

* A palavra const cria uma constante que não pode ser alterada. 


const double PI = 3.14;

Console.WriteLine(PI);

Saída:

3.14

Dicas importantes 💡

📌 C# exige tipo da variável
⚠ Toda linha termina com ;
🔤 string usa aspas duplas
🔡 char usa aspas simples
🚀 Muito usado em jogos e empresas
🧠 Linguagem fortemente tipada
