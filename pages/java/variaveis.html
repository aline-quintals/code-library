Variáveis em Java

Variáveis são espaços de memória utilizados para armazenar informações durante a execução do programa.

O que são variáveis?

Em Java, variáveis servem para guardar valores que podem ser utilizados ao longo do código.

Cada variável possui:

  • ✔ Um tipo
  • ✔ Um nome
  • ✔ Um valor

Regras das variáveis

O Java possui algumas regras importantes para criar variáveis corretamente.

  • ✔ O nome não pode começar com número
  • ✔ Java diferencia letras maiúsculas e minúsculas
  • ✔ Sempre finalize linhas com ;
  • ✔ O tipo da variável deve ser informado

Tipos mais utilizados

🔤

String

Armazena textos.

🔢

int

Armazena números inteiros.

📈

double

Armazena números decimais.

✔

boolean

Armazena true ou false.

String

* A variável String é utilizada para armazenar textos. Os textos precisam estar entre aspas duplas. 


String nome = "Aline";

System.out.println(nome);

Saída:

Aline

Inteiro

* O tipo int é utilizado para armazenar números inteiros, sem casas decimais. 


int idade = 20;

System.out.println(idade);

Saída:

20

Double

* O tipo double é utilizado para números decimais, como altura, peso e valores monetários. 


double altura = 1.70;

System.out.println(altura);

Saída:

1.7

Boolean

* O tipo boolean armazena apenas dois valores: true (verdadeiro) ou false (falso). 


boolean aprovado = true;

System.out.println(aprovado);

Saída:

true

Char

* O tipo char armazena apenas um único caractere. Diferente da String, utiliza aspas simples. 


char letra = 'A';

System.out.println(letra);

Saída:

A

Alterando valores

* As variáveis podem receber novos valores durante a execução do programa. 


String cidade = "São Paulo";

cidade = "Rio de Janeiro";

System.out.println(cidade);

Saída:

Rio de Janeiro

Concatenando textos

* A concatenação junta textos utilizando o símbolo +. 


String nome = "Aline";

System.out.println("Olá, " + nome);

Saída:

Olá, Aline

Constantes com final

* A palavra final cria uma constante, ou seja, um valor que não pode ser alterado. 


final double PI = 3.14;

System.out.println(PI);

Saída:

3.14

Dicas importantes 💡

📌 Java exige o tipo da variável
⚠ Toda linha termina com ;
🔤 String usa aspas duplas
🔡 char usa aspas simples
🚀 Variáveis ajudam a organizar dados
📚 Java é muito utilizado no mercado
