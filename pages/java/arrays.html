Arrays em Java

Arrays armazenam múltiplos valores.

O que são arrays?

Arrays guardam vários elementos do mesmo tipo.

  • ✔ Armazenam múltiplos dados
  • ✔ Possuem índices
  • ✔ Podem ser percorridos com loops
  • ✔ Facilitam organização de dados

Estrutura de um array

No Java, é necessário informar o tipo de dado do array. 


String[] nomes = {

    "Ana",
    "Carlos",
    "Aline"

};

Nesse exemplo, o array armazena apenas textos.

Array simples

* Os arrays começam na posição 0. 


String[] frutas = {

    "Maçã",
    "Banana",
    "Uva"

};

System.out.println(frutas[0]);

Saída:

Maçã

Mostrando várias posições

* Cada posição do array possui um índice diferente. 


String[] cores = {

    "Azul",
    "Verde",
    "Vermelho"

};

System.out.println(cores[0]);
System.out.println(cores[1]);
System.out.println(cores[2]);

Saída:

Azul
Verde
Vermelho

Loop no array

* O for percorre automaticamente todos os elementos do array. 


String[] nomes = {

    "Ana",
    "Carlos",
    "Aline"

};

for (String nome : nomes) {

    System.out.println(nome);

}

Saída:

Ana
Carlos
Aline

Array de números

* Arrays também podem armazenar números inteiros. 


int[] numeros = {

    10,
    20,
    30,
    40

};

System.out.println(numeros[2]);

Saída:

30

Alterando valores

* Os valores do array podem ser alterados. 


String[] jogos = {

    "Minecraft",
    "Roblox",
    "Free Fire"

};

jogos[1] = "Fortnite";

System.out.println(jogos[1]);

Saída:

Fortnite

Tamanho do array

* O length mostra quantas posições existem no array. 


String[] animais = {

    "Cachorro",
    "Gato",
    "Coelho"

};

System.out.println(animais.length);

Saída:

3

Array com FOR tradicional

* O for tradicional utiliza índices para percorrer o array. 


String[] frutas = {

    "Maçã",
    "Banana",
    "Uva"

};

for (int i = 0; i < frutas.length; i++) {

    System.out.println(frutas[i]);

}

Saída:

Maçã
Banana
Uva

Dicas importantes 💡

  • ✔ Arrays começam no índice 0
  • ✔ Arrays armazenam apenas um tipo de dado
  • ✔ Use length para descobrir o tamanho
  • ✔ O for ajuda a percorrer arrays
  • ✔ Arrays ajudam na organização dos dados
  • ✔ É possível alterar valores pelas posições
