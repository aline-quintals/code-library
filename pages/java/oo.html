Orientação a Objetos

Java é uma linguagem fortemente orientada a objetos.

O que é POO?

POO organiza o código utilizando objetos e classes.

  • ✔ Classes
  • ✔ Objetos
  • ✔ Métodos
  • ✔ Atributos
  • ✔ Encapsulamento

Conceitos importantes

A Programação Orientada a Objetos possui conceitos fundamentais.

📦

Classe

Modelo utilizado para criar objetos.

🧍

Objeto

Instância criada a partir da classe.

⚙️

Método

Ação que o objeto pode executar.

🏷️

Atributo

Características do objeto.

Criando uma classe

* Uma classe funciona como um modelo para criar objetos. 


class Pessoa {

    String nome = "Aline";

}

Pessoa pessoa = new Pessoa();

System.out.println(pessoa.nome);

Saída:

Aline

Atributos

* Atributos armazenam informações sobre o objeto. 


class Carro {

    String modelo = "Ferrari";

    int ano = 2025;

}

Carro carro = new Carro();

System.out.println(carro.modelo);
System.out.println(carro.ano);

Saída:

Ferrari
2025

Método

* Métodos representam ações que o objeto pode executar. 


class Cachorro {

    void latir() {

        System.out.println("Au au!");

    }

}

Cachorro dog = new Cachorro();

dog.latir();

Saída:

Au au!

Método com parâmetro

* Métodos podem receber informações através de parâmetros. 


class Pessoa {

    void mostrarNome(String nome) {

        System.out.println(nome);

    }

}

Pessoa pessoa = new Pessoa();

pessoa.mostrarNome("Aline");

Saída:

Aline

Construtor

* O construtor é executado automaticamente quando o objeto é criado. 


class Pessoa {

    String nome;

    Pessoa(String nome) {

        this.nome = nome;

    }

}

Pessoa pessoa = new Pessoa("Carlos");

System.out.println(pessoa.nome);

Saída:

Carlos

Criando vários objetos

* Uma mesma classe pode criar vários objetos. 


class Aluno {

    String nome;

}

Aluno aluno1 = new Aluno();
Aluno aluno2 = new Aluno();

aluno1.nome = "Ana";
aluno2.nome = "Lucas";

System.out.println(aluno1.nome);
System.out.println(aluno2.nome);

Saída:

Ana
Lucas

Usando THIS

* O this representa o próprio objeto. 


class Produto {

    String nome;

    void definirNome(String nome) {

        this.nome = nome;

    }

}

Produto produto = new Produto();

produto.definirNome("Notebook");

System.out.println(produto.nome);

Saída:

Notebook

Exemplo na vida real 🌎

Na Programação Orientada a Objetos, podemos transformar objetos da vida real em código.

Por exemplo:

  • 🚗 Um carro pode possuir: modelo, cor e velocidade
  • 🐶 Um cachorro pode possuir: nome, raça e idade
  • 👤 Uma pessoa pode possuir: nome, idade e altura

Essas características são chamadas de atributos.

Já as ações desses objetos são chamadas de métodos.

  • 🚗 Carro → acelerar()
  • 🐶 Cachorro → latir()
  • 👤 Pessoa → falar()

Exemplo completo

* Esse exemplo cria um carro com atributos e um método que representa uma ação. 


class Carro {

    String modelo = "Ferrari";

    void acelerar() {

        System.out.println("O carro acelerou!");

    }

}

Carro carro = new Carro();

System.out.println(carro.modelo);

carro.acelerar();

Saída:

Ferrari
O carro acelerou!

Dicas importantes 💡

  • ✔ Classes funcionam como modelos
  • ✔ Objetos são criados com new
  • ✔ Métodos representam ações
  • ✔ Atributos armazenam dados
  • ✔ O construtor inicializa objetos
  • ✔ O this representa o próprio objeto
  • ✔ Java é fortemente orientado a objetos
