Funções em Java

Métodos ajudam a reutilizar códigos.

O que são funções?

Funções em Java são chamadas de métodos.

  • ✔ Organização
  • ✔ Reutilização
  • ✔ Código limpo
  • ✔ Automatização de tarefas

Estrutura de um método

Os métodos possuem tipo de retorno, nome e parâmetros. 


public static void saudacao() {

    System.out.println("Olá!");

}

Nesse exemplo:

  • ✔ public → acesso público
  • ✔ static → pode ser chamado sem criar objeto
  • ✔ void → não retorna valor
  • ✔ saudacao → nome do método

Método simples

* Métodos ajudam a executar tarefas sempre que forem chamados. 


public static void saudacao() {

    System.out.println("Olá!");

}

saudacao();

Saída:

Olá!

Método com parâmetro

* Parâmetros permitem enviar informações para dentro do método. 


public static void mostrarNome(String nome) {

    System.out.println(nome);

}

mostrarNome("Aline");

Saída:

Aline

Método com retorno

* O return devolve um valor para o programa. 


public static int somar(int a, int b) {

    return a + b;

}

System.out.println(somar(5, 3));

Saída:

8

Método com boolean

* Métodos também podem retornar verdadeiro ou falso. 


public static boolean maiorDeIdade(int idade) {

    return idade >= 18;

}

System.out.println(maiorDeIdade(20));

Saída:

true

Método com Scanner

* O Scanner permite receber dados digitados pelo usuário. 


import java.util.Scanner;

public class Main {

    public static void mostrarMensagem(String nome) {

        System.out.println("Olá, " + nome);

    }

    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);

        System.out.print("Digite seu nome: ");

        String nome = scanner.nextLine();

        mostrarMensagem(nome);

    }

}

Exemplo:

Digite seu nome: Aline
Olá, Aline

Método sem retorno

* Métodos com void não retornam valores. 


public static void mensagem() {

    System.out.println("Bem-vindo!");

}

mensagem();

Saída:

Bem-vindo!

Chamando o método várias vezes

* Um método pode ser reutilizado quantas vezes forem necessárias. 


public static void saudacao() {

    System.out.println("Olá!");

}

saudacao();
saudacao();
saudacao();

Saída:

Olá!
Olá!
Olá!

Dicas importantes 💡

  • ✔ Métodos ajudam a evitar repetição de código
  • ✔ Use nomes claros nos métodos
  • ✔ O return devolve valores
  • ✔ O void significa sem retorno
  • ✔ Parâmetros recebem informações
  • ✔ Métodos deixam o código organizado
  • ✔ Métodos podem ser reutilizados várias vezes
