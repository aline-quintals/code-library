Condicionais em Java

Condicionais permitem que o programa tome decisões.

O que são condicionais?

As estruturas condicionais executam ações diferentes dependendo de uma condição.

  • ✔ if
  • ✔ else
  • ✔ else if
  • ✔ Operadores lógicos

Operadores de comparação

Os operadores servem para comparar valores.

  • == Igual
  • != Diferente
  • > Maior que
  • < Menor que
  • >= Maior ou igual
  • <= Menor ou igual

If

* O if executa um bloco de código caso a condição seja verdadeira. 


int idade = 18;

if (idade >= 18) {

    System.out.println("Maior de idade");

}

Saída:

Maior de idade

If e Else

* O else é executado quando a condição do if for falsa. 


int numero = 5;

if (numero % 2 == 0) {

    System.out.println("Par");

} else {

    System.out.println("Ímpar");

}

Saída:

Ímpar

If, Else If e Else

* O else if permite criar outras verificações no código. 


int nota = 7;

if (nota >= 9) {

    System.out.println("Excelente");

} else if (nota >= 6) {

    System.out.println("Aprovado");

} else {

    System.out.println("Reprovado");

}

Saída:

Aprovado

Verificando senha

* O método equals() é usado para comparar textos no Java. 


String senha = "java123";

if (senha.equals("java123")) {

    System.out.println("Acesso permitido");

} else {

    System.out.println("Senha incorreta");

}

Saída:

Acesso permitido

Usando AND

* O operador && significa AND. As duas condições precisam ser verdadeiras. 


int idade = 20;
boolean temIngresso = true;

if (idade >= 18 && temIngresso) {

    System.out.println("Entrada permitida");

} else {

    System.out.println("Entrada negada");

}

Saída:

Entrada permitida

Usando OR

* O operador || significa OR. Apenas uma condição precisa ser verdadeira. 


String dia = "domingo";

if (dia.equals("sábado") || dia.equals("domingo")) {

    System.out.println("Fim de semana");

} else {

    System.out.println("Dia útil");

}

Saída:

Fim de semana

Condicional com Scanner

* O Scanner permite receber dados digitados pelo usuário. 


import java.util.Scanner;

Scanner scanner = new Scanner(System.in);

System.out.print("Digite um número: ");

int numero = scanner.nextInt();

if (numero > 10) {

    System.out.println("Maior que 10");

} else {

    System.out.println("Menor ou igual a 10");

}

Exemplo:

Digite um número: 15
Maior que 10

Operador ternário

* O operador ternário é uma forma curta de escrever condicionais simples. 


int idade = 17;

String resultado = (idade >= 18)
? "Maior de idade"
: "Menor de idade";

System.out.println(resultado);

Saída:

Menor de idade

Dicas importantes 💡

  • ✔ Use parênteses nas condições
  • ✔ Use chaves { } para organizar blocos
  • ✔ O Java diferencia letras maiúsculas e minúsculas
  • ✔ Use equals() para comparar textos
  • ✔ O operador && significa AND
  • ✔ O operador || significa OR
  • ✔ A indentação ajuda na leitura do código
