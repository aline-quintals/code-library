O que é Java?

Java é uma linguagem de programação orientada a objetos criada pela Sun Microsystems e atualmente mantida pela :contentReference[oaicite:0]{index=0}.

Ela foi desenvolvida com a ideia de:

  • ✔ Rodar em diferentes sistemas operacionais
  • ✔ Ser segura e robusta

  • ✔ Possuir alta performance
