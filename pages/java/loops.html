Loops em Java

Loops executam ações repetidamente.

O que são loops?

Loops ajudam a automatizar repetições no código.

  • ✔ for
  • ✔ while
  • ✔ do while
  • ✔ break
  • ✔ continue

Para que servem?

Os loops são usados para repetir tarefas automaticamente.

  • ✔ Criar contadores
  • ✔ Percorrer arrays
  • ✔ Automatizar tarefas
  • ✔ Repetir verificações

Loop For

* O for é muito utilizado quando sabemos quantas vezes o loop irá repetir. 


for (int i = 1; i <= 5; i++) {

    System.out.println(i);

}

Saída:

1
2
3
4
5

Loop While

* O while repete o código enquanto a condição for verdadeira. 


int contador = 1;

while (contador <= 5) {

    System.out.println(contador);

    contador++;

}

Saída:

1
2
3
4
5

Loop Do While

* O do while executa o bloco pelo menos uma vez antes da verificação. 


int numero = 1;

do {

    System.out.println(numero);

    numero++;

} while (numero <= 5);

Saída:

1
2
3
4
5

Loop em Array

* O loop pode percorrer todos os itens de um array automaticamente. 


String[] frutas = {

    "Maçã",
    "Banana",
    "Uva"

};

for (String fruta : frutas) {

    System.out.println(fruta);

}

Saída:

Maçã
Banana
Uva

Usando Break

* O break interrompe o loop imediatamente. 


for (int i = 1; i <= 10; i++) {

    if (i == 5) {

        break;

    }

    System.out.println(i);

}

Saída:

1
2
3
4

Usando Continue

* O continue pula uma repetição do loop. 


for (int i = 1; i <= 5; i++) {

    if (i == 3) {

        continue;

    }

    System.out.println(i);

}

Saída:

1
2
4
5

Loop infinito

* Um loop infinito acontece quando a condição nunca se torna falsa. 


while (true) {

    System.out.println("Loop infinito");

}

Resultado:

Loop infinito
Loop infinito
Loop infinito
...

Contagem regressiva

* Loops também podem fazer contagens regressivas. 


for (int i = 5; i >= 1; i--) {

    System.out.println(i);

}

Saída:

5
4
3
2
1

Dicas importantes 💡

  • ✔ Use for quando souber o número de repetições
  • ✔ Use while para condições variáveis
  • ✔ O do while executa pelo menos uma vez
  • ✔ Use break para interromper loops
  • ✔ Use continue para pular repetições
  • ✔ Evite loops infinitos sem controle
  • ✔ A indentação ajuda a organizar o código
