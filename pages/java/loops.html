Loops em Java

Loops executam ações repetidamente.

O que são loops?

Loops ajudam a automatizar repetições no código.

  • ✔ for
  • ✔ while
  • ✔ do while

Loop For


for (int i = 1; i <= 5; i++) {

    System.out.println(i);

}

Saída:

1
2
3
4
5

Loop While


int contador = 1;

while (contador <= 5) {

    System.out.println(contador);

    contador++;

}

Saída:

1
2
3
4
5
