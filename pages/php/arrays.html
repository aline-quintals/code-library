Arrays em PHP

Arrays armazenam múltiplos valores dentro de uma única variável.

O que são arrays?

Arrays ajudam a organizar informações relacionadas, como listas de nomes, produtos, números e muito mais.

  • ✔ Armazenam múltiplos dados
  • ✔ Possuem índices
  • ✔ Facilitam organização de informações
  • ✔ Funcionam com loops
  • ✔ Podem guardar textos e números

Estrutura de um array

No PHP, os arrays são criados utilizando colchetes []. 


$frutas = [

    "Maçã",
    "Banana",
    "Uva"

];

Cada item do array possui uma posição, também chamada de índice.

Array simples

* Os arrays começam na posição 0. 


$frutas = [

    "Maçã",
    "Banana",
    "Uva"

];

echo $frutas[0];

Saída:

Maçã

Mostrando várias posições

* Cada posição do array possui um índice diferente. 


$cores = [

    "Azul",
    "Verde",
    "Vermelho"

];

echo $cores[0];
echo "
";

echo $cores[1];
echo "
";

echo $cores[2];

Saída:

Azul
Verde
Vermelho

Loop no array

* O foreach percorre automaticamente todos os itens do array. 


$nomes = [

    "Ana",
    "Carlos",
    "Aline"

];

foreach ($nomes as $nome) {

    echo $nome;
    echo "
";

}

Saída:

Ana
Carlos
Aline

Array de números

* Arrays também podem armazenar números. 


$numeros = [

    10,
    20,
    30,
    40

];

echo $numeros[2];

Saída:

30

Alterando valores

* É possível alterar valores pelas posições do array. 


$jogos = [

    "Minecraft",
    "Roblox",
    "Free Fire"

];

$jogos[1] = "Fortnite";

echo $jogos[1];

Saída:

Fortnite

Adicionando novos itens

* O PHP permite adicionar novos valores facilmente. 


$frutas = [

    "Maçã",
    "Banana"

];

$frutas[] = "Uva";

print_r($frutas);

Saída:

Array
(
    [0] => Maçã
    [1] => Banana
    [2] => Uva
)

Tamanho do array

* A função count() mostra quantos itens existem. 


$animais = [

    "Cachorro",
    "Gato",
    "Coelho"

];

echo count($animais);

Saída:

3

Array associativo

* Arrays associativos utilizam nomes ao invés de números como índice. 


$pessoa = [

    "nome" => "Aline",
    "idade" => 20,
    "cidade" => "São Paulo"

];

echo $pessoa["nome"];

Saída:

Aline

Array com FOR tradicional

* O for tradicional percorre o array utilizando índices. 


$frutas = [

    "Maçã",
    "Banana",
    "Uva"

];

for ($i = 0; $i < count($frutas); $i++) {

    echo $frutas[$i];
    echo "
";

}

Saída:

Maçã
Banana
Uva

Dicas importantes 💡

  • ✔ Arrays começam no índice 0
  • ✔ Use foreach para percorrer arrays
  • ✔ Use count() para descobrir o tamanho
  • ✔ Arrays ajudam na organização dos dados
  • ✔ Arrays associativos usam nomes como índices
  • ✔ É possível alterar e adicionar valores
  • ✔ Arrays são muito usados em sistemas web
