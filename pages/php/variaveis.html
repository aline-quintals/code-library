Variáveis em PHP

Variáveis são usadas para armazenar informações que podem ser utilizadas durante a execução do programa.

O que são variáveis?

Em PHP, variáveis são espaços na memória que guardam valores como textos, números e booleanos. Elas sempre começam com o símbolo $.

  • ✔ Não precisam de tipo fixo
  • ✔ São dinâmicas
  • ✔ Sempre começam com $

Tipos de dados

O PHP trabalha com diferentes tipos de dados automaticamente.

  • String: textos
  • Integer: números inteiros
  • Float: números decimais
  • Boolean: verdadeiro ou falso

String

* Strings armazenam textos.

Inteiro

* Inteiros representam números sem casas decimais.

Float

* Floats representam números com casas decimais.

Boolean

* Booleanos representam valores verdadeiro ou falso.

Dicas importantes 💡

  • ✔ Toda variável começa com $
  • ✔ PHP detecta o tipo automaticamente
  • ✔ Use nomes claros nas variáveis
  • ✔ Variáveis são sensíveis a maiúsculas
