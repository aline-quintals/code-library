Loops em PHP

Loops são estruturas de repetição usadas para executar um bloco de código várias vezes de forma automática.

O que são loops?

Em PHP, loops permitem repetir instruções enquanto uma condição for verdadeira ou enquanto houver elementos em uma lista para percorrer.

  • ✔ for
  • ✔ while
  • ✔ foreach
  • ✔ do while

Loop FOR

* O for é usado quando sabemos exatamente quantas vezes o loop deve repetir. 


for ($i = 1; $i <= 5; $i++) {

    echo $i;
    echo "
";

}

Loop WHILE

* O while executa enquanto a condição for verdadeira. 


$contador = 1;

while ($contador <= 5) {

    echo $contador;
    echo "
";

    $contador++;

}

Loop FOREACH

* O foreach percorre automaticamente todos os elementos de um array. 


$nomes = [

    "Ana",
    "Carlos",
    "Aline"

];

foreach ($nomes as $nome) {

    echo $nome;
    echo "
";

}

DO WHILE

* O do while executa pelo menos uma vez antes de verificar a condição. 


$numero = 1;

do {

    echo $numero;
    echo "
";

    $numero++;

} while ($numero <= 3);

Dicas importantes 💡

Os loops são fundamentais para automatizar tarefas e evitar repetição de código manual.

  • ✔ Use for quando souber o número de repetições
  • ✔ Use while quando depender de condição
  • ✔ Use foreach para arrays
  • ✔ Use do while quando precisar executar pelo menos uma vez
  • ✔ Evite loops infinitos sem controle
