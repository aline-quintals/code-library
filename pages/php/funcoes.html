Condicionais em PHP

Condicionais permitem que o sistema tome decisões com base em condições.

O que são condicionais?

As condicionais verificam se algo é verdadeiro ou falso para executar diferentes ações.

  • ✔ if
  • ✔ else
  • ✔ elseif
  • ✔ switch
  • ✔ Operadores lógicos

Operadores de comparação

  • == Igual
  • != Diferente
  • > Maior que
  • < Menor que
  • >= Maior ou igual
  • <= Menor ou igual

If

* O if executa o bloco apenas se a condição for verdadeira. 


$idade = 18;

if ($idade >= 18) {

    echo "Maior de idade";

}

Saída:

Maior de idade

If e Else

* O else executa quando a condição for falsa. 


$numero = 5;

if ($numero % 2 == 0) {

    echo "Par";

} else {

    echo "Ímpar";

}

Saída:

Ímpar

If, Elseif e Else

* O elseif permite verificar várias condições. 


$nota = 7;

if ($nota >= 9) {

    echo "Excelente";

} elseif ($nota >= 6) {

    echo "Aprovado";

} else {

    echo "Reprovado";

}

Saída:

Aprovado

Usando AND

* O operador and exige que todas as condições sejam verdadeiras. 


$idade = 20;
$temIngresso = true;

if ($idade >= 18 and $temIngresso) {

    echo "Entrada permitida";

} else {

    echo "Entrada negada";

}

Saída:

Entrada permitida

Usando OR

* O operador or exige apenas uma condição verdadeira. 


$dia = "domingo";

if ($dia == "sábado" or $dia == "domingo") {

    echo "Fim de semana";

} else {

    echo "Dia útil";

}

Saída:

Fim de semana

Switch

* O switch facilita verificações com vários casos. 


$cor = "azul";

switch ($cor) {

    case "azul":
        echo "Cor azul";
        break;

    case "verde":
        echo "Cor verde";
        break;

    default:
        echo "Outra cor";

}

Saída:

Cor azul

Dicas importantes 💡

  • ✔ Condições retornam verdadeiro ou falso
  • ✔ Use chaves para organizar o código
  • ✔ O elseif verifica múltiplas condições
  • ✔ O switch deixa o código mais organizado
  • ✔ Operadores lógicos ajudam nas verificações
  • ✔ Condicionais são muito usadas em sistemas
